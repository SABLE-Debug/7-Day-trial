— 02 / The loop

Seven days. Five Looms.

One target, one Loom per day. Hard stop at end of Day 7 — no extensions, no pity Looms.

Day 1Score 50 prospects. Pick five.
Day 2Six-hour dossiers. Record Looms 1–2.
Day 3Send 1–2 via LinkedIn. Record 3.
Day 4Send 3 via X. Record 4.
Day 5Send 4 via email. Record 5.
Day 6Send 5. Run Template C ladder.
Day 7Read the scoreboard. Pivot or stay.
— 03 / The decision

At Day 7. One read.

The reply count is the only signal that matters. Two real replies and a booked call validates the pivot. Anything less, tighten or stay.

2+ replies · 1+ call · 1+ proposal
Fully commit to the SaaS pivot. Retarget Path E content. Rebuild Chapter 11.
1 reply · no call
Run a second 7-day test with a tighter ICP. Do not pivot yet.
0 replies
Stay AKHARA med-spa positioning. Adopt the MOBIUS visual rebrand only.
