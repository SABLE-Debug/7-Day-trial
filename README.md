# MOBIUS

**Productized design subscription for funded brands.**

A senior designer on retainer for less than half what an in-house junior costs. Cancel anytime.

## THE OFFER

| Tier | Price | What |
|---|---|---|
| Starter | $1,995 / mo | 1 active request · 72-hour turnaround · cancel anytime |
| **Standard** | **$3,995 / mo** | **Unlimited requests · 48-hour turnaround · pause or cancel anytime** |
| Brand sprint | $7,500 one-time | 14-day rebrand · fixed scope · subscription discount after |

Aesthetic: editorial minimalism that moves. Dark mode native. Fraunces serif + Geist Mono. Peach gradient on void.

ICP: **funded brands** that want **editorial design with motion**. Industry-agnostic. The aesthetic filters the rest.
